# 项目说明

对 CT 扫描得到的颗粒图像进行的相关研究，主要涉及 CT 图像处理、颗粒几何形态分析、深度学习模型生成颗粒、离散元 clump 生成算法。

## 文件（夹）说明

-   `clump/`: 离散元 clump 生成算法
-   `mindboggle/`: 网上的开源代码，与计算 Zernike 矩相关
-   `nn/`: 神经网络模型
-   `utils/`: 一些工具类／工具函数
